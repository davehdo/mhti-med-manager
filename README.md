# Introduction
This is a starter template that include the following stack

## Server side
* Rails 4.2.4
* Mongodb

## Client side
* Angular
* Jquery
* Coffee

## Styling
* Sass
* Bootstrap
* Fontawesome ```<i class="fa fa-camera-retro"></i>```


# Getting Started
